# How to start #

### What is this repository for? ###

vue.js로 작성된 gs 뷰입니다.

### How do I get set up? ###

```bash
git clone https://github.com/yesfordev/internVue.git

cd internVue

npm install

npm run serve
```

### 진행 상황 ###

Home화면에는 만든 사람 이름만 나와 있는 것이 정상입니다.
구현된 view를 확인하기 위해서는 왼쪽 navBar의 DataHub -> 한국은행 지표 관리 -> GDP 로 이동하셔서 확인해주시면 됩니다.
